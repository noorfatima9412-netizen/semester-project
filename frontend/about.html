About This Project

What is this system?

The Pharmacy Management System is a full-stack web application built for managing daily pharmacy operations. It helps pharmacy staff track medicine inventory, handle customer records, process sales, and manage prescriptions digitally.

Problem It Solves

  • Manual paper-based records are slow and error-prone
  • Hard to track medicine stock and expiry dates
  • No central place for sales and customer data
  • Difficult to get quick statistics for the pharmacy

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: FastAPI (Python)
  • Database: PostgreSQL on Supabase
  • Authentication: JWT Token
  • Deployment: Render (backend) + Vercel/Netlify (frontend)
